# Helm (optional)

Package, install and upgrade applications with the Helm package manager.

## Commands
| Command | Description |
| --- | --- |
| helm repo add | add a chart repository |
| helm install/upgrade/uninstall | manage releases |
| helm rollback | roll back a release |
---

## Tasks
### 1. Install Helm and add a chart repository
### 2. Install a chart as a release
### 3. Override values
### 4. Upgrade and then roll back the release
