# ConfigMaps and Secrets

Separate configuration and sensitive data from images; inject them as env vars or files.

## Commands
| Command | Description |
| --- | --- |
| kubectl create configmap | create a ConfigMap |
| kubectl create secret | create a Secret |
---

## Tasks
### 1. Create a ConfigMap from literals and from a file
### 2. Inject ConfigMap values as environment variables
### 3. Mount a ConfigMap as files in a Pod
### 4. Create a Secret and use it as env vars / a mounted file
