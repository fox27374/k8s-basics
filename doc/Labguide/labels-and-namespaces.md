# Labels and namespaces

Organise and select resources with labels; isolate them with namespaces.

## Commands
| Command | Description |
| --- | --- |
| kubectl label | add/remove labels |
| kubectl get -l | select resources by label |
| kubectl create namespace | create a namespace |
