# Deployments

Run and manage Pods at scale with a Deployment and its ReplicaSet.

## Commands
| Command | Description |
| --- | --- |
| kubectl create deployment | create a Deployment |
| kubectl scale | change the replica count |
| kubectl get rs | list ReplicaSets |
| kubectl rollout status | watch a rollout |
